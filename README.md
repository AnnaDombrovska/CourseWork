# Travel Assistant Telegram Bot 

A full-cycle Telegram bot developed in **C#** that helps users discover travel destinations, hotels, and local attractions by integrating real-time data from the **TripAdvisor API**.

## About the Project
This project demonstrates the complete software development lifecycle (SDLC), from initial requirements gathering and architecture design to the final release. It showcases practical skills in consuming external APIs, managing relational databases, and building interactive chat interfaces.

## Tech Stack & Tools
*   **Language:** C#
*   **External API:** TripAdvisor API (RESTful integration, JSON parsing)
*   **Database:** PostgreSQL
*   **Frameworks/Libraries:** Telegram.Bot API
*   **Architecture:** Object-Oriented Programming (OOP) principles

## Key Features & Development Highlights
*   **Full-Cycle Development:** Independently designed the business logic, data models, and API communication flows.
*   **TripAdvisor Integration:** Configured HTTP requests to fetch and process live data regarding locations and tourist spots.
*   **Database Management:** Structured and connected a **PostgreSQL** database for secure storage of user interactions, session states, and query history.
*   **Asynchronous Processing:** Utilized async/await patterns in C# to ensure smooth and non-blocking interactions for users.

## AI-Assisted Development
As part of an active experiment with AI-assisted programming, Large Language Models (ChatGPT, Claude) were utilized as "pair programmers" for:
*   Designing the core backend architecture and database schema.
*   Writing and optimizing PostgreSQL queries.
*   Generating data models to accurately deserialize JSON responses from the TripAdvisor API.
*   Debugging API connection flows and structuring the bot's dialogue logic.
